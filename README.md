## 🚀 Project Overview

The **lab-natty-or-not** project, developed for the **DIO (Digital Innovation One)** laboratory, explores the frontier of **Generative AI**. The goal is to create realistic, high-quality content that challenges the boundary between human creation and AI generation—the "Natty" (Natural) vs. "Fake" debate. 

This project focuses on the practical and ethical application of foundation models to enhance creativity and professional productivity.

[Image of Generative AI workflow: Input -> Prompt Engineering -> Foundation Model -> Human Review -> Output]

## 🛠️ Tech Stack & AI Tools

* **ChatGPT (OpenAI):** Utilized for advanced text generation, creative storytelling, and content structuring.
* **Amazon Bedrock:** Explored for its enterprise-grade access to diverse foundation models (FMs) like Claude, Llama, and Titan.
* **AWS PartyRock:** Used for rapid prototyping of AI-driven applications and hands-on experimentation with prompt flows.
* **Markdown:** For structured documentation and clean technical presentation.

---

## 🧐 The "Natty" Creation Process

To ensure the content felt natural and original, I followed a rigorous **Prompt Engineering** workflow:
1. **Contextualization:** Defining the persona and objective of the generation.
2. **Iterative Prototyping:** Using Amazon Bedrock/PartyRock to test different foundation models.
3. **Refinement:** Applying the "Human-in-the-loop" principle to review, edit, and ensure ethical alignment.
4. **Final Polish:** Adjusting tone, flow, and coherence to achieve a "Natty" result.
